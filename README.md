Welcome to the GitHub repository for AyikeLoremaster's supplemental worldbuilding related to ZPD Chronicles! Feel free to make a pull request and make your own lore inspired by what's here!

File summary below:

ayike map.ora: Master multi-layer image file that can be opened via paint,net.
Bottom layer is the locations map from EUV, a grand strategy game made by Paradox Interactive.
The 2nd layer is a basic land-and-water layer.
The 3rd layer has the geographic extents of all 288 countries, on land.
The 4th layer has the labels for the countries, for wasteland areas, and for major bodies of water.
The 5th layer is blank, and can be used to help plan any edits someone opening the file may wish to use.

ayike map.png: Single layer png file that shows all 288 countries, with labels.

timeline.txt: Text document detailing over 600 years of geopolitical history. Has plenty of opprtunity for more detail.

README.md: You're looking at it right now. Self-explanatory.
